# Strategic Planner & Media Planner Portfolio

This is a professional portfolio website for Worraphon Arunsamran (Amber), a Strategic Planner & Media Planner. The website showcases expertise in strategic communication, consumer insight, media planning, and trend analysis.

## Features

- Responsive design that works on all device sizes
- Light/dark mode toggle
- Professional portfolio showcasing case studies
- Services page with pricing
- Resume page
- Contact form
- Modern, clean aesthetic

## Project Structure

```
├── index.html              # Main landing page
├── resume.html             # Resume page
├── styles.css              # CSS styles
├── script.js               # JavaScript functionality
├── qwen.md                 # Project requirements
└── README.md               # This file
```

## Technologies Used

- HTML5
- CSS3 (with custom properties for theming)
- JavaScript (ES6+)
- Font Awesome for icons
- Responsive design with Flexbox and Grid

## Responsive Design

The website is fully responsive and adapts to different screen sizes:
- Desktop: 1200px and above
- Tablet: 768px to 1199px
- Mobile: Below 768px

## Browser Compatibility

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Deployment

### Deploy to Vercel

1. Push your code to a GitHub repository
2. Go to [Vercel](https://vercel.com)
3. Connect your GitHub account
4. Import your repository
5. Deploy!

### Deploy to Netlify

1. Push your code to a GitHub repository
2. Go to [Netlify](https://netlify.com)
3. Connect your GitHub account
4. Import your repository
5. Deploy!

### Local Development

1. Clone or download the repository
2. Open `index.html` in your browser

## Customization

To customize this website for your own use:

1. Update the content in all HTML files (name, bio, case studies, etc.)
2. Replace the placeholder images with your own
3. Update the contact information
4. Modify colors in `styles.css` by adjusting the CSS custom properties
5. Update the meta tags with your own information

## Color Theme

The website uses CSS custom properties for easy theme switching:

Light theme:
- Primary: #2563eb (blue)
- Secondary: #1e40af (dark blue)
- Accent: #f59e0b (amber)

Dark theme:
- Primary: #3b82f6 (blue)
- Secondary: #2563eb (dark blue)
- Accent: #f59e0b (amber)
